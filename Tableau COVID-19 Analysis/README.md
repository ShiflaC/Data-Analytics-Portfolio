# Tableau Project – COVID‑19 Cases & Deaths Analysis

## Overview
This project analyzes global COVID‑19 case and death trends using Tableau. The dataset contains daily reported values across multiple countries and WHO regions. The dashboard visualizes historical patterns in new and cumulative cases and deaths, helping understand how the pandemic evolved over time.

## Dataset
Dialy COVID-19 data including:
- Date
- Country
- WHO Region
- New Cases and Deaths
- Cumulative Cases and Deaths

## Steps Completed
- Imported the COVID‑19 dataset into Tableau  
- Cleaned and prepared the data  
- Created time‑series charts for new cases and deaths
- Added filters for country and region  
- Designed an interactive dashboard with KPIs and trend visuals  

## Skills Used
- Tableau data cleaning  
- Time‑series visualization  
- Dashboard design and layout
