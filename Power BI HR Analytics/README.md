## Overview
A Power BI dashboard analyzing employee attrition using an HR dataset from Kaggle.  
The goal is to identify why employees leave and what factors influence turnover.

## Dataset
- Source: Kaggle (HR Analytics dataset)
- Includes: Age, Gender, Job Role, Salary, Performance, Attrition, etc.

## Tools Used
- Power BI Desktop
- Power Query
- DAX

## Steps Taken
- Cleaned and transformed data in Power Query
- Built data model with relationships
- Created DAX measures for KPIs
- Designed interactive dashboard with slicers and visuals

## Key Insights
- Higher attrition among younger employees
- Low job satisfaction linked to higher turnover
- Certain job roles show higher risk of leaving
